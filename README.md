## How run the project

Clone the project :

```
git clone https://github.com/renansouza12/FUYU-landing-page.git

```
Enter the project folder and open it:
```
cd FUYU-landing-page

code .
```
Type ```npm install``` to install node packages and ```ng serve ``` to run a localhost server.
